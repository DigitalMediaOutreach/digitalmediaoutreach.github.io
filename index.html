DIGITAL MEDIA OUTREACH

Introducing kids to the exciting world of physical computing, programming and interaction design!
Workshops


memory

Physical Computing

We teach kids the basics of physical computing through the use of MakeyMakey. Kids are able to draw and design their own game controller and physically attach it to a circuit.

code

Programming

What's more fun than playing games? Making them of course! We offer a few games to help kids learn instructional based programming like loops and conditional statements without typing or coding.

mode_edit

Design

Kids are able to realize their imagination by designing their own game assets which can be implemented in game for them to play with.
